- Add/edit/delete lost & found items

## How to Run Locally
1. Clone repo  
2. `npm install`  
3. Set up `.env` file  
4. `npm start`  

## Additional Feature
**GitHub OAuth Login** - Users can log in using their GitHub accounts.
